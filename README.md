## ADC 2024 Talk — Virtual Analog Modelling

Presentation delivered at the Audio Developer Conference (ADC) 2024.

This project aims to digitally replicate the electrical behavior of the MXR Distortion+ guitar pedal circuit. The session discusses virtual analog (VA) modelling techniques, DSP design considerations, and implementation strategies for audio software application.
